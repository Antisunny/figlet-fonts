This comment was included with the font file and propably contains additional information about the font:
Modified by Paul Burton September 96 from caligraphy.flf as a demonstration of 
FigWin features.  Modification took about 5 minutes using FigWin's "rectangle 
search and replace" feature.  The characters previously consisted of only "*" 
characters.  These changes required two steps:

First, all instances of "*" were replaced with "#" to give the font a darker 
appearance.  

                         #                      /
Then, all instances of: #  were replaced with: /   (Each of these is a 
rectangle of four squares, including two blanks.)

Also added a period, exclamation point, and question mark and changed the 
default smushmode.

Original caligraphy.flf composed by Vinney Thai October 94
